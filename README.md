## DMS-UPLOAD

**用于接受DMS的动态表单上传文件**

## 使用

### 安装
```bash
> git clone https://github.com/win-winFE/dms-upload.git
> yarn
```

### 配置Redis
```bash
# vim app/util/constants.js
```

### 启动
```bash
> yarn dev
```
